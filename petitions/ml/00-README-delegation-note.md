# ചിറ്റൂർ നിവേദക സംഘം — തിരുവനന്തപുരം, 2026 ജൂലൈ 15

**സംഘം:** ചിറ്റൂർ അഗ്രികൾച്ചർ മാർക്കറ്റിംഗ് സൊസൈറ്റി, കൊഴിഞ്ഞാമ്പാറ (ചിറ്റൂർ താലൂക്ക്, പാലക്കാട് ജില്ല) യുടെ പ്രതിനിധികൾ.

**അകമ്പടി:** അഡ്വ. സുമേഷ് അച്യുതൻ, എം.എൽ.എ. — ചിറ്റൂർ (മണ്ഡലം നം. 58), കേരള നിയമസഭ.

**ഉദ്ദേശ്യം:** ചിറ്റൂർ നിയോജക മണ്ഡലത്തെ, വിശേഷിച്ച് **എരുത്തേമ്പതി, വടകരപ്പതി, കൊഴിഞ്ഞാമ്പാറ** എന്നീ മഴനിഴൽ പഞ്ചായത്തുകളെ ബാധിക്കുന്ന അഞ്ചു വിഷയങ്ങളിൽ ബന്ധപ്പെട്ട മന്ത്രിമാർക്ക് നിവേദനങ്ങൾ സമർപ്പിച്ച് നടപടികൾ വേഗത്തിലാക്കാൻ അഭ്യർത്ഥിക്കുക.

## വസ്തുതകളെക്കുറിച്ചുള്ള കുറിപ്പ്

ഈ കരടുകൾ പൊതുവായി ലഭ്യമായ വിവരങ്ങൾ (കേന്ദ്ര ഭൂജല ബോർഡ്, കേരള ഭൂജല വകുപ്പ്, ജലവിഭവ വകുപ്പ്/KIIDC, Milma/MRCMPU, CPCB/TNPCB, രാസവള വകുപ്പ്) അടിസ്ഥാനമാക്കി തയ്യാറാക്കിയവയാണ്. ചില വസ്തുതകൾ തിരുത്തിയിട്ടുണ്ട്:

- **മഴ** അളക്കുന്നത് **മില്ലിമീറ്ററി (mm)** ലാണ്, സെന്റിമീറ്ററിലല്ല. ചിറ്റൂരിന്റെ മഴനിഴൽ മേഖലയിൽ പ്രതിവർഷം ഏകദേശം **≤1,000 mm** മാത്രം; കേരള ശരാശരി ഏകദേശം **3,000 mm** — അതായത് മൂന്നിലൊന്ന്.
- പരാമർശിക്കുന്ന കുപ്പിവെള്ള വിവാദം **പ്ലാച്ചിമട (പെരുമാട്ടി പഞ്ചായത്ത്) കൊക്കക്കോള** കേസാണ്.
- **PAP** കണക്കുകൾ രേഖാമൂലമുള്ള അവകാശം (മണക്കടവ് വിയറിൽ **7.25 TMC / 7,250 mcft**) അനുസരിച്ചാണ്.

ഓരോ ഇംഗ്ലീഷ് നിവേദനത്തിലും കൃത്യമായ **ജി.ഒ./നിയമ പരാമർശങ്ങളും** മന്ത്രിക്ക് പുറപ്പെടുവിക്കാവുന്ന ഒരു **മാതൃകാ കരട് സർക്കാർ ഉത്തരവും (Draft G.O.)** ഉൾപ്പെടുത്തിയിട്ടുണ്ട്. ലെറ്റർഹെഡിൽ അച്ചടിക്കുന്നതിനു മുമ്പ് `[...]` എന്ന ബ്രാക്കറ്റുകൾ പൂരിപ്പിക്കുകയും അനുബന്ധങ്ങൾ ചേർക്കുകയും ചെയ്യുക.
