# சிற்றூர் மனுக்குழு — திருவனந்தபுரம், 15 ஜூலை 2026

**குழு:** சிற்றூர் விவசாய சந்தைப்படுத்தல் சங்கம், கொழிஞ்சாம்பாறை (சிற்றூர் வட்டம், பாலக்காடு மாவட்டம்) பிரதிநிதிகள்.

**உடன் வருபவர்:** வழக்கறிஞர் சுமேஷ் அச்சுதன், எம்.எல்.ஏ. — சிற்றூர் (தொகுதி எண். 58), கேரள சட்டமன்றம்.

**நோக்கம்:** சிற்றூர் சட்டமன்றத் தொகுதியை, குறிப்பாக **எருத்தேம்பதி, வடகரப்பதி, கொழிஞ்சாம்பாறை** ஆகிய மழை நிழல் பஞ்சாயத்துகளை பாதிக்கும் ஐந்து பிரச்சினைகளில் சம்பந்தப்பட்ட அமைச்சர்களுக்கு மனுக்களை சமர்ப்பித்து நடவடிக்கைகளை விரைவுபடுத்த வேண்டுகோள் விடுத்தல்.

## தகவல்கள் குறித்த குறிப்பு

இந்த வரைவுகள் பொதுவில் கிடைக்கும் தரவுகளை (மத்திய நிலத்தடி நீர் வாரியம், கேரள நிலத்தடி நீர்த்துறை, நீர்வளத் துறை/KIIDC, Milma/MRCMPU, CPCB/TNPCB, உரத்துறை) அடிப்படையாகக் கொண்டவை. சில தகவல்கள் திருத்தப்பட்டுள்ளன:

- **மழை** அளக்கப்படுவது **மில்லிமீட்டரில் (mm)**, சென்டிமீட்டரில் அல்ல. சிற்றூரின் மழை நிழல் மண்டலத்தில் ஆண்டுக்கு சுமார் **≤1,000 mm** மட்டுமே; கேரள சராசரி சுமார் **3,000 mm** — அதாவது மூன்றில் ஒரு பங்கு.
- குறிப்பிடப்படும் குடிநீர் தொழிற்சாலை சர்ச்சை **பிளாச்சிமடா (பெருமாட்டி பஞ்சாயத்து) கோகோ-கோலா** வழக்கு.
- **PAP** எண்ணிக்கைகள் ஆவணப்படுத்தப்பட்ட உரிமையின்படி (மணக்கடவு தடுப்பணையில் **7.25 TMC / 7,250 mcft**).

ஒவ்வொரு ஆங்கில மனுவிலும் சரியான **அரசாணை/சட்டக் குறிப்புகளும்**, அமைச்சர் வெளியிடக்கூடிய ஒரு **மாதிரி வரைவு அரசாணையும் (Draft G.O.)** சேர்க்கப்பட்டுள்ளன. லெட்டர்ஹெட்டில் அச்சிடும் முன் `[...]` அடைப்புகளை நிரப்பி இணைப்புகளைச் சேர்க்கவும்.
